0. node 설치 (16.19 로 하니 에러가 안남)
1. vue cli

```
npm install -g @vue/cli
# OR
yarn global add @vue/cli
```

2. scss

```
npm install --save-dev node-sass sass-loader
```

3. vue-router

```
npm install vue-router@next
```
